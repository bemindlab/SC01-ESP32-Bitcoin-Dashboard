## Quick Reference

### Hardware Specs (from Datasheet)
- **MCU:** ESP32-S3-WROVER-N16R2
- **Flash:** 16MB SPI Flash
- **PSRAM:** 2MB PSRAM
- **Display:** 3.5" IPS TFT, 320x480 resolution, ST7796 driver
- **Touch:** FT6X36 capacitive touch controller (I2C)
- **Connectivity:** Wi-Fi 802.11 b/g/n (2.4GHz), Bluetooth 5.0
